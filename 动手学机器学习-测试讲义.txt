动手学机器学习测试讲义

来源说明
本讲义根据本地文件“动手学机器学习 (张伟楠等) .pdf”的目录、内容提要和可识别章节线索整理，用于测试智慧教育智能体平台的课程资料入库、RAG 问答、知识点总结和知识图谱生成。讲义为教学化改写，不逐字复制教材正文。

课程定位
本课程以“原理理解 + 代码实践 + 实验反思”为主线，帮助学习者从机器学习基础概念出发，逐步掌握参数化模型、非参数化模型和无监督模型。学习时不要把算法当成孤立名称，而要同时说明任务目标、输入输出、模型假设、优化目标、评价指标和常见失败原因。

课程总目标
1. 能解释机器学习的基本定义：系统基于经验数据提升特定任务性能。
2. 能区分预测、决策、监督学习、无监督学习和强化学习的任务边界。
3. 能用向量、矩阵、梯度、凸函数等数学工具描述模型。
4. 能完成 KNN、线性回归、逻辑斯谛回归、神经网络、SVM、决策树、集成学习、k 均值、PCA、概率图模型、EM、自编码器等模型的基本实现或调用。
5. 能根据数据特点选择模型，并用欠拟合、过拟合、正则化、交叉验证、评价指标解释模型表现。
6. 能将教材中的算法原理对应到 Python Notebook 或 sklearn/PyTorch 实验流程。

四大部分导学
第一部分：机器学习基础
包括初探机器学习、数学基础、K 近邻、线性回归、机器学习基本思想。重点建立机器学习问题的基本框架：数据、模型、损失、优化、泛化、评估。

第二部分：参数化模型
包括逻辑斯谛回归、双线性模型、神经网络与多层感知机、卷积神经网络、循环神经网络。重点理解模型参数如何通过损失函数训练得到，以及深度网络如何通过层次表示完成复杂任务。

第三部分：非参数化模型
包括支持向量机、决策树、集成学习与梯度提升决策树。重点理解间隔、划分规则、模型组合、Boosting 迭代纠错等思想。

第四部分：无监督模型
包括 k 均值聚类、主成分分析、概率图模型、EM 算法、自编码器。重点理解无标签数据中的结构发现、低维表示、概率建模和隐变量估计。

第1章 初探机器学习
学习目标
理解人工智能与机器学习的关系，说明机器学习为什么能通过经验数据提升任务性能。

核心知识点
1. 人工智能可从任务角度理解为预测与决策，从技术角度理解为搜索、推理、学习、博弈等能力。
2. 机器学习关注基于数据自动优化任务表现，常见预测任务包括分类、回归、聚类和生成。
3. 泛化能力是机器学习奏效的关键，模型不能只记住训练样本，还要能处理未见样本。
4. 归纳偏置体现模型的先验假设，例如线性模型偏好线性关系，KNN 偏好相似样本标签相近。
5. 机器学习存在限制：数据偏差、任务定义不清、评价指标不完整、模型不可解释或部署环境变化都会导致失效。

课堂引导
教师可先让学生列举生活中的智能系统，再追问这些系统依赖什么数据、优化什么指标、可能犯什么错误。

练习建议
请用“经验、性能、任务”三个词解释机器学习，并举一个校园场景中的机器学习例子。

易错提醒
不要把人工智能和机器学习完全等同。机器学习是人工智能的重要技术路径之一，但人工智能还包括搜索、推理、博弈等方法。

第2章 机器学习的数学基础
学习目标
掌握机器学习中常用的向量、矩阵、梯度、凸函数概念，为后续模型推导做准备。

核心知识点
1. 向量表示样本、参数或特征，向量维度对应特征数量或表示空间维度。
2. 矩阵可表示数据集、线性变换、参数集合或协方差结构。
3. 矩阵运算帮助描述批量样本预测、线性方程组求解和模型参数更新。
4. 梯度表示函数在各变量方向上的变化率，是参数优化的核心工具。
5. 凸函数具有较好的优化性质，理解凸性有助于分析损失函数和收敛行为。

课堂引导
将一个二维样本点、一批样本矩阵和线性回归参数写成数学符号，再解释每个符号在代码中的形状。

练习建议
给定一个 3 个样本、2 个特征的数据矩阵，写出线性模型的预测表达式，并说明参数维度。

易错提醒
不要只背矩阵公式，要同时检查维度是否匹配。很多机器学习代码错误本质上是张量维度错误。

第3章 K 近邻算法
学习目标
理解 KNN 的相似性假设，能用 KNN 完成分类和回归任务。

核心知识点
1. KNN 根据样本之间的距离寻找最近邻，用投票或平均得到预测结果。
2. 距离度量决定“相似”的含义，常见距离包括欧氏距离、曼哈顿距离等。
3. K 值影响模型复杂度，K 太小容易受噪声影响，K 太大可能忽略局部结构。
4. 特征尺度会影响距离计算，因此归一化或标准化通常很重要。
5. KNN 训练过程简单，但预测时需要与大量样本比较，数据量大时效率会成为问题。

课堂引导
让学生手算一个二维平面上的 KNN 分类例子，再改变 K 值观察分类结果变化。

练习建议
用 sklearn 实现 KNN 分类，并比较不同 K 值和是否标准化对准确率的影响。

易错提醒
不要忽视特征尺度。例如身高和收入如果直接放在一起计算距离，量纲较大的特征可能主导结果。

第4章 线性回归
学习目标
掌握线性回归的模型形式、学习目标、解析解和梯度下降求解思想。

核心知识点
1. 线性回归假设输出是输入特征的线性组合。
2. 常用平方损失衡量预测值和真实值之间的误差。
3. 解析方法可通过正规方程求解参数，但在高维或大规模数据中可能代价较高。
4. 梯度下降通过迭代方式更新参数，适合更一般的优化场景。
5. 学习率过大可能震荡或发散，学习率过小会导致收敛缓慢。

课堂引导
从一元线性回归图像出发，说明斜率和截距如何影响预测线，再推广到多特征情况。

练习建议
分别用解析解、手写梯度下降和 sklearn LinearRegression 求解同一个回归任务，比较参数和误差。

易错提醒
线性回归的“线性”通常指参数线性，不一定意味着只能处理原始特征的直线关系，可以通过特征变换表达非线性模式。

第5章 机器学习的基本思想
学习目标
理解欠拟合、过拟合、正则化、特征相似度、参数、超参数、数据集划分和交叉验证。

核心知识点
1. 欠拟合表示模型能力不足或训练不足，训练集和测试集表现都差。
2. 过拟合表示模型过度适应训练集噪声，训练集表现好但测试集表现差。
3. 正则化通过约束模型复杂度提升泛化能力，常见形式包括 L1 和 L2。
4. 参数由训练数据学习得到，超参数需要人为设置或通过验证集选择。
5. 数据集通常划分为训练集、验证集和测试集，测试集不能被反复用于调参。
6. 交叉验证适合样本量较小时估计模型稳定性。

课堂引导
展示同一数据集上不同复杂度模型的拟合曲线，让学生判断欠拟合和过拟合。

练习建议
在回归任务中加入多项式特征，观察模型复杂度、训练误差、测试误差之间的关系。

易错提醒
不要把验证集和测试集混用。验证集用于模型选择，测试集用于最终评估。

第6章 逻辑斯谛回归
学习目标
理解逻辑斯谛函数、最大似然估计、分类评价指标、交叉熵与最大似然之间的关系。

核心知识点
1. 逻辑斯谛回归将线性得分通过 Sigmoid 函数转化为类别概率。
2. 它名称中有“回归”，但常用于二分类或多分类问题。
3. 最大似然估计从概率角度寻找最能解释观测标签的参数。
4. 分类评价指标包括准确率、精确率、召回率、F1、AUC 等。
5. 交叉熵损失可以看作最大似然估计在分类任务中的优化形式。

课堂引导
用一个二分类样本解释线性得分、Sigmoid 概率、阈值决策和交叉熵损失。

练习建议
手写逻辑斯谛回归训练过程，并与 sklearn LogisticRegression 的结果比较。

易错提醒
不要只看准确率。类别不平衡时，准确率可能掩盖模型对少数类的糟糕表现。

第7章 双线性模型
学习目标
理解矩阵分解和因子分解机，掌握它们在推荐和稀疏特征建模中的作用。

核心知识点
1. 矩阵分解将用户-物品评分矩阵分解为低维隐向量表示。
2. 隐向量可以捕捉用户偏好和物品特征之间的潜在关系。
3. 因子分解机适合处理高维稀疏特征，并能建模二阶特征交互。
4. 双线性思想关注两个向量或两个特征集合之间的交互得分。
5. 推荐任务中需要关注冷启动、稀疏性、过拟合和评价指标。

课堂引导
用“用户喜欢的电影类型”和“电影隐含风格”解释隐向量的含义。

练习建议
构造一个小型评分矩阵，尝试用矩阵分解补全缺失评分。

易错提醒
隐向量不是人工指定的显式标签，而是模型从数据中学到的低维表示。

第8章 神经网络与多层感知机
学习目标
理解人工神经网络、感知机、隐含层、多层感知机、反向传播和 PyTorch 实现流程。

核心知识点
1. 神经网络由层、权重、偏置和激活函数组成。
2. 感知机是早期线性分类模型，多层感知机通过隐含层和非线性激活增强表达能力。
3. 反向传播利用链式法则高效计算各层参数梯度。
4. 常见激活函数包括 Sigmoid、Tanh、ReLU 等。
5. PyTorch 训练流程通常包括定义模型、定义损失、选择优化器、前向传播、反向传播和参数更新。

课堂引导
让学生画出一个输入层、隐含层、输出层的网络，并标注每一步张量形状。

练习建议
用 PyTorch 实现一个多层感知机完成简单分类任务，记录训练损失和验证准确率。

易错提醒
没有非线性激活的多层线性网络本质上仍可等价为一个线性变换。

第9章 卷积神经网络
学习目标
理解卷积、图像分类、预训练 CNN、内容表示和风格表示。

核心知识点
1. 卷积通过局部连接和权重共享提取空间局部特征。
2. CNN 适合处理图像，因为图像具有局部相关性和平移结构。
3. 卷积层、池化层、激活函数和全连接层共同构成典型图像分类网络。
4. 预训练网络可作为特征提取器，提升小数据场景下的效果。
5. 风格迁移通常分别建模内容表示和风格表示。

课堂引导
用一个小矩阵演示卷积核滑动计算过程，再解释卷积核如何检测边缘或纹理。

练习建议
用预训练 VGG 或其他 CNN 提取图像特征，并观察不同层特征的差异。

易错提醒
卷积核不是手工固定规则，深度学习中多数卷积核参数通过训练得到。

第10章 循环神经网络
学习目标
理解 RNN 的序列建模思想和门控循环单元 GRU 的基本作用。

核心知识点
1. RNN 通过隐藏状态在时间步之间传递历史信息。
2. 序列任务包括文本、语音、时间序列预测等。
3. 普通 RNN 可能出现梯度消失或梯度爆炸。
4. GRU 通过门控机制控制信息保留和更新，缓解长期依赖问题。
5. 序列模型训练时需要关注输入序列长度、批处理方式和隐藏状态初始化。

课堂引导
用一句话的词序列解释隐藏状态如何逐步吸收上下文信息。

练习建议
手写一个简化 RNN 或 GRU 的前向传播过程，说明每个门的作用。

易错提醒
RNN 不是只看当前输入，它的输出依赖当前输入和历史隐藏状态。

第11章 支持向量机
学习目标
理解 SVM 的数学描述、序列最小优化、核函数和 sklearn 中的 SVM 工具。

核心知识点
1. SVM 的核心思想是寻找最大间隔分类超平面。
2. 支持向量是决定分类边界的关键样本。
3. 软间隔允许少量分类错误，通过惩罚参数平衡间隔和误差。
4. 核函数将低维非线性问题转化为高维空间中的线性可分问题。
5. SMO 是求解 SVM 对偶问题的一类有效方法。

课堂引导
在二维平面画出两类点、分隔线、间隔和支持向量，让学生解释为什么最大间隔有助于泛化。

练习建议
比较线性核、多项式核和 RBF 核在同一分类任务上的表现。

易错提醒
核函数并不是显式把所有样本坐标都算到高维空间，而是通过核技巧计算内积。

第12章 决策树
学习目标
理解决策树构造、ID3、C4.5、CART、剪枝和 sklearn 决策树工具。

核心知识点
1. 决策树通过递归划分特征空间形成一系列判断规则。
2. ID3 常用信息增益选择划分特征。
3. C4.5 使用增益率缓解信息增益偏向取值多的特征的问题。
4. CART 可用于分类和回归，常用基尼指数或平方误差作为划分准则。
5. 剪枝用于降低过拟合风险，提高泛化能力。

课堂引导
用“是否适合户外运动”的小数据集手工构建一棵决策树。

练习建议
用 sklearn 训练决策树，观察最大深度、叶子节点最小样本数对模型表现的影响。

易错提醒
决策树可解释性强，但如果不限制深度或不剪枝，很容易过拟合训练数据。

第13章 集成学习与梯度提升决策树
学习目标
理解自举聚合、随机森林、集成学习器、提升算法、AdaBoost 和梯度提升。

核心知识点
1. 集成学习通过组合多个基学习器提升整体性能。
2. Bagging 通过自举采样训练多个模型，主要降低方差。
3. 随机森林在 Bagging 基础上加入特征随机性，提升多样性。
4. Boosting 逐步关注前一轮错误样本或残差，主要降低偏差。
5. GBDT 使用梯度方向拟合损失函数残差，是常用强模型。

课堂引导
用“多个学生投票做题”和“错题逐轮加强训练”区分 Bagging 与 Boosting。

练习建议
比较单棵决策树、随机森林和 GBDT 在同一数据集上的训练误差和测试误差。

易错提醒
集成不是简单堆模型，基学习器之间的差异性和组合方式决定了集成效果。

第14章 k 均值聚类
学习目标
理解 k 均值聚类原理、动手实现过程和 k-means++ 初始化思想。

核心知识点
1. k 均值通过“分配样本到最近中心”和“更新聚类中心”交替迭代。
2. 目标是最小化簇内样本到中心的平方距离。
3. k 值需要根据业务目标、评价指标或经验选择。
4. 初始中心会影响结果，k-means++ 通过更合理初始化降低不稳定性。
5. k 均值偏好球形簇，对异常点和尺度敏感。

课堂引导
让学生在二维点图上手动执行两轮 k 均值迭代，观察中心变化。

练习建议
对二维数据使用不同 k 值聚类，并用轮廓系数或可视化结果比较。

易错提醒
聚类结果没有天然标签，评价时要结合任务目标，不要把簇编号当作真实类别。

第15章 主成分分析
学习目标
理解主成分、方差最大化、特征分解和 PCA 的 sklearn 实现。

核心知识点
1. PCA 寻找能够保留数据最大方差的正交方向。
2. 主成分是新的低维坐标轴，按解释方差大小排序。
3. 特征分解可用于求解协方差矩阵的主方向。
4. PCA 可用于降维、去噪、可视化和降低后续模型复杂度。
5. PCA 是无监督方法，不直接利用类别标签。

课堂引导
在二维散点图中画出最大方差方向，解释为什么沿该方向投影能保留更多信息。

练习建议
对高维数据做 PCA 降到二维，并观察不同类别在二维空间中的分布。

易错提醒
方差最大不等于分类最优。PCA 关注数据整体变化方向，不保证类别最容易分开。

第16章 概率图模型
学习目标
理解贝叶斯网络、最大后验估计、朴素贝叶斯文本分类、马尔可夫网络和图像去噪示例。

核心知识点
1. 概率图模型用图结构表达变量之间的条件依赖关系。
2. 贝叶斯网络是有向图，适合表达因果或条件生成关系。
3. 最大后验估计在似然基础上加入先验信息。
4. 朴素贝叶斯基于条件独立假设，常用于文本分类。
5. 马尔可夫网络是无向图，适合表达局部相互作用。

课堂引导
用“疾病、症状、检测结果”画一个简单贝叶斯网络，说明条件概率如何更新信念。

练习建议
实现一个朴素贝叶斯文本分类器，比较不同平滑参数对结果的影响。

易错提醒
朴素贝叶斯的“朴素”来自条件独立假设，该假设简化计算，但不一定完全符合真实数据。

第17章 EM 算法
学习目标
理解高斯混合模型、EM 算法的 E 步和 M 步、一般形式与收敛性。

核心知识点
1. EM 适合含有隐变量或缺失变量的概率模型参数估计。
2. E 步根据当前参数估计隐变量后验分布。
3. M 步在隐变量分布固定的情况下更新模型参数。
4. 高斯混合模型可用 EM 估计各高斯成分的均值、协方差和混合权重。
5. EM 通常保证似然不下降，但可能收敛到局部最优。

课堂引导
用“混合了两类身高数据但不知道每个人类别”的例子解释隐变量。

练习建议
用 GMM 拟合二维数据分布，观察不同初始化对结果的影响。

易错提醒
EM 不是一次求出全局最优，而是在隐变量估计和参数更新之间迭代改进。

第18章 自编码器
学习目标
理解自编码器结构、编码器、解码器、重构损失和表示学习用途。

核心知识点
1. 自编码器由编码器和解码器组成，目标是重构输入。
2. 编码器把输入压缩为低维表示，解码器从表示恢复输入。
3. 重构损失衡量输入和输出之间的差异。
4. 自编码器可用于降维、去噪、异常检测和表示学习。
5. 如果模型容量过大且没有约束，自编码器可能学成简单复制函数。

课堂引导
把自编码器类比为“压缩-解压”系统，讨论什么样的压缩表示才有学习价值。

练习建议
训练一个简单自编码器重构手写数字图像，观察隐空间维度对重构质量的影响。

易错提醒
自编码器虽然不需要标签，但仍需要设计合适的瓶颈、正则化或噪声机制，否则学不到有用表示。

阶段性测验建议
1. 概念题：解释机器学习中的泛化能力，并说明训练误差和测试误差的区别。
2. 计算题：给定一组样本，手算一次 KNN 分类或一次线性回归梯度更新。
3. 分析题：某模型训练集准确率 99%、测试集准确率 70%，判断问题并给出改进方案。
4. 实践题：用 sklearn 分别训练逻辑斯谛回归、决策树、随机森林，比较指标并解释差异。
5. 开放题：选择一个真实问题，说明适合使用监督学习、无监督学习还是强化学习，并给出理由。

学习路径建议
1. 第1-2章：先建立机器学习定义和数学语言。
2. 第3-5章：通过 KNN、线性回归和模型评估理解数据、模型、损失、泛化。
3. 第6-10章：学习参数化模型和神经网络，重点连接公式、代码和训练流程。
4. 第11-13章：学习非参数化模型与集成思想，重点比较模型假设和泛化机制。
5. 第14-18章：学习无监督模型，重点理解没有标签时如何发现结构或表示。

常见薄弱点归因
1. 不会调模型，常见根因是没有区分参数和超参数。
2. 不会解释测试集表现差，常见根因是没有理解过拟合、数据泄漏和交叉验证。
3. 不会写代码实现，常见根因是没有把数学符号对应到数组形状、训练循环和优化器。
4. 不会选择模型，常见根因是没有分析任务类型、数据规模、特征形态和评价指标。
5. 不会理解无监督方法，常见根因是把聚类编号、主成分方向或隐变量误当成真实标签。
